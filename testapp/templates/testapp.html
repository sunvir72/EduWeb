Welcome to EduWeb

An educational portal for students and teachers

Some Key features of the website are:

Courses

- Teachers can easily create new courses
- They can enroll students in them, upon which an automated course invite is sent to each student
-Students on the other hand can register for the courses that they are enrolled for

Content

- Teachers can upload course contents which the students can access at any time. Some of these are:
> Announcements
> Reference links
> Assignments
> Practice questionnaires and Tests

Questionnaires

- Teachers can easily create questionnaires
- Each question can have positive and negative marks and upto six options
- Time limit can also be set for them
- Questions can either be added one by one or bulk uploaded through a csv file
- These can be monitored and Email reminders can be sent to students who have not yet attempted them
- When a student attempts a test, the screen goes into fullscreen mode
- The test would be submitted automatically if the student tries to exit fullscreen

Machine learning

- Every activity of student is monitored and stored in the backend to create datasets
- Teachers can work on these datasets and appy machine learning models
- They can even upload their own dataset and work on it
